# URL Shortener Microservice

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Custom Logging API Integration

## Features

- Shorten long URLs
- Custom shortcodes
- Default & custom expiry
- Redirection
- Stats with click logs
- Remote logging

## How to Run

1. Clone the repo
2. Run `npm install`
3. Create `.env` file
4. Start server with `npm start`
